## 📌 Sobre

A **Brasil GO** é uma plataforma web para quem quer viajar pelo Brasil de forma prática e acessível. O usuário pode buscar destinos, comparar opções, fazer reservas e gerenciar suas viagens em um só lugar.

Projeto final do curso de Front-End — **Programa PROA + Serasa Experian**, desenvolvido pela turma Grupo 4 em 2026.

---

## 🗂️ Páginas

- 🏠 **Home** — apresentação da plataforma e destaque dos diferenciais
- 🔐 **Login** — acesso à conta do usuário
- 📝 **Cadastro** — criação de nova conta
- ✈️ **Menu de Viagens** — listagem de destinos disponíveis
- 📋 **Minhas Viagens** — histórico de viagens marcadas e canceladas
- 👤 **Perfil** — visualização e edição dos dados cadastrais
- 🔍 **Busca** — pesquisa personalizada de destinos
